How to run pelican

Sun 18 March 2018
By Hyunmin
  • local 환경에서 실시간 페이지 보기
$ fab serve
  • markdown 변경파일을 실시간으로 html 형식으로 변경하기
$ fab rebuild
  • github page에 적용하기
$ make github or fab gh_pages
